CoffeeBeanz

Project that entails creating a functioning ecommerce website using mainly PHP, SQL, HTML and CSS. The website allows allows users to make purchases of a variety of coffee beans. With the option of making an account to view order status.

My responsibilties included the following:
Implementing a functional employee dashboard that allows employees to update their personal information, and give them access to the current product inventory as well as updating customer orders.
Implementing a functional customer dashboard that allows the user to see their orders and details of those orders, including the price of the order, order status, card used to make purchase, etc.
